Unsematic CSS Grid by Nathan Smith

The grid this template uses is the one produced by the amazing Nathan Smith. The grid is called Unsemantic and it is the successor to the 960.gs (for those who know that). Unsemantic is a fluid grid system that is the successor to the 960 Grid System. It works in a similar way, but instead of being a set number of columns, it's entirely based on percentages. For any other information, documentation and known issues, you can refer directly to the Unsemantic website. There you'll finde anything you may need.
